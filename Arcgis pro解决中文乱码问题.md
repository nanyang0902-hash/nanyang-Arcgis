### 方法一：添加 `.cpg` 索引文件（最推荐，彻底解决） 🌟

ArcGIS 需要一个小文件来告诉它：“嘿，这个文件是用中文编码写的，请用中文方式打开。”

1. **打开文件夹**：在 Windows 资源管理器中，找到存放你这个 `Shapefile.shp` 的文件夹。
    
2. **新建文本文件**：
    
    - 在文件夹空白处右键 -> **新建** -> **文本文档**。
        
    - 将文件名修改为与你的 Shapefile 主文件名完全一致，但后缀名改为 `.cpg`。
        
    - _例如：如果你的文件叫 `Result.shp`，那这个新文件必须叫 `Result.cpg`。_
        
3. **写入代码页**：
    
    - 右键点击这个 `.cpg` 文件，选择用 **记事本** 打开。
        
    - 在里面输入数字 **`936`**（这是简体中文 GBK 的代码）。
        
    - 保存并关闭。
        
4. **重启加载**：
    
    - 关闭并重新打开 ArcGIS Pro（或者移除图层重新添加）。
        
    - 再次打开属性表或符号系统，乱码应该就变成正常的中文了。
